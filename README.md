Upcoming tablature displaying application built with `Flask`/`SQLAlchemy` for backend and `React`

- `Flask`/`SQLAlchemy`/`unittest` for backend
- `React` for frontend
- `AWS` for hosting (WIP)
- `Travis CI` for CI/CD (WIP)
